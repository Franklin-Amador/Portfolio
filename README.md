# 🚀 Portfolio - Franklin Amador

Portfolio personal desarrollado con Astro, React y Tailwind CSS. Sitio web moderno, optimizado y completamente responsivo para mostrar mis proyectos y habilidades como Front-end Developer.

## ✨ Características

- 🎨 **Diseño Moderno**: Interfaz limpia y atractiva con gradientes personalizados
- 🌓 **Dark/Light Mode**: Sistema de temas con toggle interactivo
- 📱 **Totalmente Responsivo**: Optimizado para todos los dispositivos
- ⚡ **Rendimiento Óptimo**: Construido con Astro para máxima velocidad
- 🎯 **SEO Optimizado**: Meta tags completos y Open Graph
- 🎭 **Animaciones Suaves**: Transiciones y efectos visuales elegantes
- 📧 **Sección de Contacto**: Múltiples formas de comunicación
- 💼 **Portfolio Interactivo**: Showcase de proyectos con tecnologías

## 🛠️ Tecnologías

- **Framework**: Astro 4.6.1
- **UI Library**: React 18.2.0
- **Styling**: Tailwind CSS 3.4.3
- **Icons**: Iconify
- **Fonts**: Google Fonts (Montserrat)
- **SEO**: Astro Sitemap

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Franklin-Amador/Portfolio.git

# Navegar al directorio
cd Portfolio

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 🚀 Comandos Disponibles

| Comando         | Acción                                                    |
| --------------- | --------------------------------------------------------- |
| `npm run dev`   | Inicia el servidor de desarrollo en `localhost:4321`     |
| `npm run build` | Construye el sitio para producción en `./dist/`          |
| `npm run preview` | Vista previa del build localmente                       |

## 🎨 Personalización

### Colores

Los colores se definen en `src/styles/global.css` usando variables CSS. Puedes modificarlas según tu preferencia.

### Proyectos

Actualiza tus proyectos en `src/components/data/portfolioData.js`

### Información Personal

Modifica tu información en:

- Hero: `src/components/hero/Hero.astro`
- About: `src/components/About/About.astro`
- Social Links: `src/components/SocialIcons.astro`
